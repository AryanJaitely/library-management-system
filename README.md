# Library Management System

A backend Library Management System built using FastAPI, MySQL, and SQLAlchemy. The project helps manage books, students, book issuance, returns, and overdue fine tracking.

## Features

* Manage student records
* Manage book records
* Manage authors and categories
* Issue books to students
* Return books and update availability
* Track due dates
* Calculate overdue fines
* Search books by title, author, or category
* RESTful API design

## Tech Stack

* FastAPI
* MySQL
* SQLAlchemy
* Pydantic

## Database Design

The database is designed using relational database concepts and includes the following entities:

* Students
* Books
* Authors
* Categories
* Issued Books

The project uses primary keys, foreign keys, and relationships to maintain data integrity and consistency.

## Concepts Used

* Database Design
* Normalization
* CRUD Operations
* SQL Joins
* Relationships
* REST APIs
* SQLAlchemy ORM
* Transaction Management

## Purpose

This project was developed to practice DBMS concepts and backend development by building a real-world library management application.
